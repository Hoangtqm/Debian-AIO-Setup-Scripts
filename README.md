# 🛠️ Cài đặt & chạy

````bash
git clone https://github.com/Hoangtqm/Debian-AIO-Setup-Scripts.git

cd Debian-AIO-Setup-Scripts

chmod +x ./AIO-Debian-Setup.sh

./AIO-Debian-Setup.sh
````
⌨️ Điều khiển menu

Phím	Chức năng

↑ / ↓	Di chuyển

SPACE	Chọn / bỏ chọn

A	Chọn tất cả

N	Bỏ chọn tất cả

ENTER	Xác nhận
